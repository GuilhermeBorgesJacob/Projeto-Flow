# Projeto-Flow
O Projeto Flow é um dos projetos do Grupo de Pesquisa e Desenvolvimento Aeroespacial (GPDA) da UFABC e tem como objetivo oferecer um meio pelo qual os estudantes possam desenvolver suas habilidades técnicas e sociais de forma prática. As pricipais áreas de estudo que o projeto engloba são as áreas de CFD e aerodinâmica.

Observações os solvers atuais utilizam a biblioteca Eigen para solucionarem sistemas lineares. O link para o download da biblioteca https://libeigen.gitlab.io/
